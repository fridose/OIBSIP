# TASK 1 – Unit Converter Application

A simple Android app that converts values between length, weight, and temperature using a clean spinner-based interface.
## Demo Video



https://github.com/user-attachments/assets/64dd5f4e-8bb6-4321-b398-d52bdcbac1b2


## Features

- Numeric input field for the value to convert.
- Category selection for Length, Weight, and Temperature.
- Source and target unit dropdowns.
- Convert button to display the result.
- Input validation with Toast messages for empty or invalid input.
- Dynamic unit lists that change based on the selected category.

## Tech Stack

- Kotlin
- Android Studio
- XML layout
- AndroidX widgets

## How It Works

1. Enter a numeric value.
2. Choose a category.
3. Select the source and target units.
4. Tap Convert.
5. The converted result is shown on screen.

## Supported Units

### Length
- Centimetre
- Metre
- Inch

### Weight
- Gram
- Kilogram
- Pound

### Temperature
- Celsius
- Fahrenheit
- Kelvin

## Notes

- The unit dropdowns update automatically when the category changes.
- Toast messages are shown if the input is empty or not a valid number.
